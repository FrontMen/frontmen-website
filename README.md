# Frontmen.nl
[Work in progress]
New version of frontmen.nl.

https://frontmen.github.io/frontmen.nl/

## Build
Generates the entire site.

`npm run build`

## Serve
Runs local server with watchers for development.

`npm run start`

## Publish
Generates the entire site and commits it to the gh-pages branch.

`npm run build`
